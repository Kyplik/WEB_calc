# WEB_calc
Необходимо реализовать браузерное приложение "Калькулятор", которое должно 
уметь:
* Сложение
* Вычитание
* Умножение
* Деление
* Возведение в степень
* Извлечение квадратного корня
* Сброс ввода
* Отмена вычислений

UI приложения должен выглядеть как стандартный калькулятор (windows, android, 
ios etc).
